# Claude UI/UX Review Prompt

Use this prompt with sanitized screenshots or local dashboard excerpts only.

```text
You are reviewing a public-safe portfolio dashboard for a Datadog detection
engineering project. The audience is recruiters and security engineering
interviewers. Please critique the UI/UX, visual hierarchy, dashboard flow,
clarity of detection coverage, and whether the design communicates practical
SOC/detection-engineering skill.

Constraints:
- No private tenant details are included.
- The dashboard must feel like a real operational security review surface, not a
  marketing page.
- It should be readable in 60 seconds by a recruiter and defensible in a deep
  technical interview.
- Suggest concrete layout, copy, and prioritization improvements.
```
